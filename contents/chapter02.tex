\chapter{材料與方法}
本章需詳述研究設計與執行細節，確保他人能驗證或重現，提供研究的可信度與科學基礎。

\section{內容要素:}
研究構想與原理:
闡述方法的理論基礎，如物理原理或數學模型。
實驗設計:
\begin{itemize}
    \item 描述硬體、軟體、材料來源、樣品製備、測量流程等細節。
   \item 數據分析：說明數據處理與分析方法，如統計工具或演算法。
   \item 方法合理性與新穎性：解釋方法選擇的理由及其創新之處。
   \item 視覺輔助：使用示意圖、流程圖或表格清晰呈現方法。

\end{itemize}

\section{撰寫建議：}
\begin{itemize}
    \item 內容具體且可重現，避免模糊描述。
    \item 若方法改進自文獻，需明確說明改進點並引用來源。
    \item 針對複雜方法，可分小節撰寫，確保邏輯清晰。
    \item 對於理論或質性研究，可詳細描述理論框架、資料來源或分析流程。
\end{itemize}
