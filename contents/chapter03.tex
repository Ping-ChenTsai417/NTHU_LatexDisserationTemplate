\chapter{結果}

本章專注呈現研究數據與分析成果，透過圖表輔助說明，作為論文的實證基礎。
\section{內容要素：}
\begin{itemize}
    \item 數據呈現：按邏輯順序展示重要數據，輔以圖表（如曲線圖、柱狀圖）。
    \item 圖表說明：每張圖表需有清晰標題與說明，確保獨立可讀。
    \item 成果介紹：依圖表順序逐一描述數據，突出與研究目標相關的發現。
    \item 重大成就：強調研究的關鍵成果及其對研究問題的回應。

\end{itemize}

這裡測試插入參考資訊~\cite{doi:10.1021/jp211439d,PhysRevB.83.075403}。

\section{撰寫建議：}
\begin{itemize}
    \item 僅呈現數據，避免過多解釋或推論（留待討論章節）。
    \item 圖表設計規範（清晰標籤、適當比例），與正文一致。
    \item 按研究目標或假設順序組織數據，增強邏輯性。
    \item 簡述數據的可靠性和再現性（如重複實驗次數或誤差範圍）。
\end{itemize}
